\section{Systemvorraussetzungen:}

Das CTMS läuft auf jedem Rechner, auf dem Java 1.4.2 und MySQL 4.1 läuft. Diese Software müssen Sie auch vorher installieren (Hinweis: Eine Anleitung zur Installation von MySQL findet sich im Anhang). Für die Sprachausgabe benötigen Sie noch eine Soundkarte. Das Installationsarchiv umfasst 50 MB; jedoch wird der Speicherbedarf vor allem durch die Daten bestimmt, die Sie im CTMS hinterlegen.

\section{Installation}
Die Installation benötigt nur wenige Schritte nachdem die oben genannten Grundlage geschaffen wurden. Die einzelnen Schritte werden im Folgenden beschrieben.

\subsection{Entpacken des Archivs}
Das Programm wird in Form eine Zip-Archivs ausgeliefert. Entpacken Sie dieses in ein beliebiges Verzeichnis.

\subsection{OpenGL-Vorbereitung}
CTMS verwendet gl4java als Schnittstelle zur Grafikausgabe. Um mit ihrem Betriebssystem interagieren zu können, müssen deshalb einige Dateien in ihre Java-Installation kopiert werden. Unter Windows wird von uns die Datei libGL4Java2.8.0.8-Win32-x86 mitgeliefert. Die darin enthaltenen Dateien müssen sie in das bin-Unterverzeichnis ihres Java Runtime Enviroments entpacken. Dieses liegt unter einem Pfad, der ähnlich zu folgendem ist: c:/Programme/Java/jdk1.5.0_05/jre/bin 

\subsection{Erstellen der Datenbank}
CTMS arbeitet mit eine MySQL-Datenbank. Diese muss vorher von einem Datenbankadministrator installiert werden. Dazu wird das SQL-Skript Create.sql mitgeliefert. Der Administrator kann damit die Tabellen einer entsprechenden Datenbank anlegen. Anschließend muss er Ihnen noch die Datei config/jdbc.properties im Installationsverzeichnis anpassen. Dort müssen folgende Angaben gemacht werden.

\begin{enumerate}
\item jdbc.DriverName: Der Name des Treibers (hier bleibt in der Regel die Voreinstellung erhalten)
\item jdbc.ConnectionURL: Der Ort der Datenbank. Hier ist insbesondere ctms durch den Namen ihrer Datenbank zu ersetzen.
\item jdbc.ConnectionUserName: Der Login des verwendeten Datenbankbenutzers.
\item jdbc.ConnectionPassword: Das Passwort zum Datenbankzugriff.
\end{enumerate}


\subsection{Starten der Applikation}
CTMS wird durch einen Doppelklick auf die Datei ctms.bat gestartet. In der jetzigen Version wird für den Administrator kein Passwort überprüft, da noch keine Passwortstrategie festgelegt wurde. Sie können sich also immer als Administrator einloggen und dann andere Benutzer anlegen.
